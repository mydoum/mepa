Requirements
---------------
* JDK 8
* Maven >= 3.3.9

How to run it
---------------
* Compile the whole project:
```
mvn clean install
```
* Run Tomcat server:
```
mvn tomcat7:run -pl mepa-front
```
* Open `http://localhost:8080/` in your favorite web browser
